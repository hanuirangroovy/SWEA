# 2027. 대각선 출력하기



## 제출1

```
# 가로 세로 같은 줄일 때만 '#' 출력되게!!
for a in range(5):
    for b in range(5):
        if a == b:
            print('#', end= '')
        else:
            print('+', end= '')
    print()
```
